شبیه‌سازی درگاه پرداخت سپ

این پروژه یک شبیه‌سازی فرانت‌اند از **درگاه پرداخت سپ** است که در ایران استفاده می‌شود. این پروژه رابط کاربری و جریان تعاملات یک درگاه پرداخت واقعی را شبیه‌سازی می‌کند و با استفاده از  HTML، CSS (SASS)  و JavaScript ایجاد شده است. هدف از این پروژه تمرین توسعه وب فرانت‌اند است.

 ویژگی‌ها
-  طراحی واکنش‌گرا : رابط کاربری به طور کامل واکنش‌گرا نیست و به اندازه‌های صفحه نمایش موبایل و دسکتاپ واکنش گرا است . ( پیاده سازی واکنش¬گرایی برای تمامی سیستم ها پیاده نشده است )

از آنجایی که این یک درگاه پرداخت است و هدف ما بیشتر بر روی استفاده از SASS بوده است ، امکاناتی همچون اعتبار سنجی فرم ، جریان پرداخت ، عناصر تعاملی JS بطور کامل پیاده سازی نشده اند زیرا ممکن است به دلیل openSource بودن این پروژه در جهت اهداف غیرقانونی مانند فیشینگ و ... از آن استفاده شود . ( این پروژه صرفاً یک پروژه آموزشی بوده است ) 



تکنولوژی‌های استفاده شده
HTML :  ساختار صفحات وب.
SASS :  برای نوشتن استایل‌ها به روشی مدولار و قابل نگهداری‌تر استفاده شده است.
JavaScript :  برای اعمال برخی از استایل ها ، همچون باز شدن acardeon ها و تایمر برنامه و نمایش لوگوی بانک براساس شماره کارت وارد شده .

Simulation of Sep Payment Gateway

This project is a front-end simulation of the  " Sep Payment Gateway " used in Iran. It mimics the
user interface and interaction flow of a real payment gateway and is created using HTML, CSS (SASS), and JavaScript.
The goal of this project is to practice front-end web development.

Features :
Responsive Design: The user interface is not fully responsive,
 but it is responsive to mobile and desktop screen sizes.
 (Responsiveness for all devices has not been fully implemented.)

Since this is a payment gateway simulation and the focus is more on using SASS,
features like form validation, payment flow, and interactive JavaScript elements have not been fully implemented.
This is because the project is open-source, and there is a potential risk that it could be misused for illegal purposes like " phishing " ,
etc. (This project is strictly for educational purposes.)

Technologies Used
HTML: Structure of the web pages.
SASS: Used for writing styles in a more modular and maintainable way.
JavaScript: Used for applying some styles, such as opening accordions,
 a program timer, and displaying the bank logo based on the entered card number.
